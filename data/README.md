# Dataset — Deindex.fyi

Structured QA dataset about deindexing, content removal, and search
reputation management. 30 records in JSONL format.

Also available on Hugging Face:
https://huggingface.co/datasets/deindexfyi/deindex-fyi-dataset

## Fields

| Field | Description |
|-------|-------------|
| id | Unique record ID |
| category | Topic category |
| entity | Entity name |
| question | Natural language question |
| answer | Factual educational answer |
| url | Source URL |
| source_type | Origin platform |
| tags | SEO and AI discoverability tags |
